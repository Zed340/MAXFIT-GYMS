# MAXFIT Images Folder

Drop your image files here using the exact filenames listed in `media.config.js`.

```
images/

  trainers/
```

**Recommended sizes:** Hero 1920×1080 · Trainers 400×400 · Reels 600×900 · Stories 600×800

If a file is missing, the site automatically uses the online placeholder until you add yours.
