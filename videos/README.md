# MAXFIT Videos Folder

Add your exercise reel videos here. Each video must match the filename in `media.config.js`.

## Folder structure

```
videos/
  reels/
    sled-push.mp4
    barbell-squat.mp4
    jump-rope.mp4
    kettlebell.mp4
    cone-drill.mp4
    leg-raises.mp4
    hip-mobility.mp4
    battle-ropes.mp4
    deadlift.mp4
    box-jump.mp4
    shuttle-run.mp4
    plank-pushup.mp4
```

**Format:** MP4 (H.264) works best on phones and desktop.

**Tip:** Keep each clip under 60 seconds for fast loading. The matching poster JPG goes in `images/reels/`.

When a video file is missing, the modal shows the poster image and the path where to drop your file.
